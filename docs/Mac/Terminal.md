# Terminal Commands
`Pwd` 当前目录  
`Echo` 复读  
`Cd /directory` 切换路径  
`Ls` = `dir`  
`Ls -r` 所有子文件夹  
`clear` 清屏  
`man` 退出 Q  
`touch -t yyyymmddhhmm` file
```shell
touch -t 199505090000
```
`Caffeinate` 不休眠  `⌃C`退出  
`banner -w 80 legolas.me` 文字  
`textutil -convert txt file`  
`Diskutil list`  
`cp` 复制文件  
`Mv` 移动文件  
`Mkdir` 创建目录  
`Cat` 文件内容
